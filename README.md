
# 🔥 Stogie Candles

**A Full-Stack Luxury Candle Brand — Where Cigar Culture Meets Craft**

Stogie Candles is a premium e-commerce brand built around the speakeasy aesthetic. Every candle captures the atmosphere of a private executive lounge — aged tobacco, bourbon, dark cedar, leather, and coffee — without the smoke.

---

## The Stack

| Layer | Technology |
|-------|-----------|
| Structure | HTML5 |
| Styling | CSS3 with custom properties for light/dark theming |
| Layout | Bootstrap 5.3 |
| Interactivity | Vanilla JavaScript |
| Payments | Stripe hosted checkout |
| Forms | Formspree |
| Design | Adobe Illustrator (logos), Adobe Photoshop (product images) |
| Fonts | Google Fonts — Cinzel + Lora |

---

## Features

- Light and dark mode with localStorage persistence
- Footer logo recolored dynamically via JavaScript canvas
- Fiery hover glow effects across all interactive elements
- Product grid with category filtering and pagination
- Responsive design — desktop, tablet, and mobile
- Zigzag editorial layout on the About page
- Consistent dark overlay on all images
- Newsletter signup and contact forms
